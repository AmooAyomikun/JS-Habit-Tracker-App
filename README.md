**Habit Tracker Web App**

Habit Tracker is a fully interactive, front-end web application designed to help users create, track, and maintain daily and weekly habits. With a visually engaging interface, users can monitor their progress, maintain streaks, and categorize habits to achieve personal goals more effectively.

**About the Project**

Building consistent habits is crucial for personal growth and productivity. This web application allows users to:

  i)Track habits on a daily or weekly basis.
  
  ii)See progress bars and streaks for motivation.
  
  iii)Categorize habits with color-coded categories and icons.
  
  iv)Persist data in LocalStorage, so no backend or server setup is required.

It’s perfect for students, professionals, and anyone looking to build meaningful routines.

**Features of the App**

**1 Add, Edit & Delete Habits:** Create new habits or modify existing ones easily.

**2 Daily & Weekly Goals:** Set habits to track on a daily or weekly basis.

**3 Progress Tracking:** Visual progress bar for each habit.

**4 Streaks:** Track current and best streaks for each habit.

**5 Categories with Icons:** Organize habits into categories like Health, Fitness, Learning, Productivity, Social, and Finance.

**6 Persistent Data:** All habits and progress are stored in LocalStorage, so data remains after refreshing the page.

**7 Responsive & Interactive UI:** Modal forms, checkboxes, and live updates enhance user experience.

**Usage Guide**
1  Click “Add Habit” to create a new habit.

2  Enter the habit name, select a category, and choose a goal type (daily/weekly).

3  If weekly, specify the frequency of completion required per week.

4  Click “Add Habit” to save. The habit appears in Pending Habits.

5  Check the checkbox on the habit card to mark it as completed.

6  Streaks and progress bars update automatically.

7  Edit or delete habits anytime using the Edit and Delete buttons.

8  All data is stored in your browser and will persist across page reloads.
